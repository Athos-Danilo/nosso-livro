# Nosso Livro — Sistema Inteligente de Biblioteca Compartilhada

Repositório central para o desenvolvimento do projeto **Nosso Livro**, um sistema baseado em microsserviços para gerenciamento e compartilhamento descentralizado de livros físicos em uma instituição.

## 📖 Documentação do Projeto

Toda a arquitetura, divisão de microsserviços por integrante e as especificações técnicas completas podem ser encontradas no arquivo:

## 🛠️ Tecnologias Principais

* **Frontend:** React (Vercel)
* **Backend (API Gateway):** Go (Render)
* **Backend (Serviços):** Go, Node.js (TypeScript) e Python (FastAPI) (Render)
* **Banco de Dados:** PostgreSQL (Neon)
* **Mensageria:** RabbitMQ (CloudAMQP)
